# Pharmacy Rebate Analytics Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze pharmacy rebate performance, drug costs, claims, and and therapeutic class trends. It enables stakeholders to monitor financial performance and drill into detailed claim-level data.

## Objectives
- Track pharmacy rebate performance
- Analyze total vs net drug costs
- Monitor rebate variance trends
- Identify top rebate-generating drugs
- Enable claim-level drill-through analysis
- To identify rebate leakage and analyze differences between actual and expected rebate amounts across drugs and therapeutic classes.

  ## Tools & Technologies
- Oracle SQL
- Power BI
- Excel / CSV
- Data Modeling
- DAX Measures

## Key Features
- Interactive slicers and filters
- KPI tracking dashboard
- Trend analysis
- Drill-through claim details
- Rebate variance monitoring

  ## KPIs
- Total Claims
- Total Cost
- Total Rebate
- Net Cost
- Rebate %
- Rebate Variance %

## Business Insights
- Immunology contributed the highest share of rebates.
- Rebate variance trends highlighted underperforming periods.
- Dashboard enabled detailed claim-level analysis.
